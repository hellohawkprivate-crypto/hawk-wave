学習ロードマップ（推奨順）

①
GitHub + Vercel で静的サイトを公開
まずpush→自動デプロイの感覚をつかむ。公開ページから始める。
②
パスワード保護ページを追加
Vercel Middlewareでシンプル認証。Sheets CSV連携も同時に試す。
③
Discord OAuth 認証を実装
NextAuth.js + Discord Provider。幹部ロール判定まで実装。
④
Streamlit でデータ分析ページを追加
施設の破壊値シミュレーター等。Streamlit Cloud で別URLとして公開。
