## 概要

Next.jsプロジェクトを初期化し、ゲームルール解説ページを作成してVercelで公開した。

---

## 実施内容

### 環境セットアップ
- Node.js を nvm で v18.13.0 → v22.22.2 にアップグレード
- `src/` に Next.js 16.2.2 を初期化（TypeScript / Tailwind CSS v4 / App Router）

### 作成したファイル

| ファイル | 内容 |
|---|---|
| `src/src/app/page.tsx` | ゲームルール解説ページ（トップページ） |
| `src/src/app/layout.tsx` | ルートレイアウト |
| `src/package.json` | Next.js依存関係 |

### ページ内容（トップページ）
- ゲーム概要（防衛側3サーバー vs 攻撃側5サーバー、破壊値5000pt）
- 施設の種類と破壊値（小型拠点〜ゴールデンパレス）
- 戦闘スケジュール（Week2〜Week4）

### Vercel連携
- GitHub連携によるCI/CD設定
- Root Directory: `src`
- 自動デプロイ（mainブランチへのpushで反映）
