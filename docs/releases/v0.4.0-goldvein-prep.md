# v0.4.0 — Goldvein War Prep アプリ

**リリース日**: 2026-04-09  
**種別**: 機能追加  
**ステータス**: 完了 ✅

---

## 概要

金脈攻防戦のイベント準備用アプリ「Goldvein War Prep」を構築。
同盟メンバー向けに、ルール解説・戦略分析・戦力分析・シミュレーターを6言語で提供する。

---

## 実施内容

### アプリ構成（7ページ）

| タブ | URL | 内容 |
|---|---|---|
| 遊び方 | `/[locale]/members/goldvein-prep/guide` | イベント概要・ゲーム紹介（吹き出しUI） |
| ルール | `/[locale]/members/goldvein-prep/rules` | 詳細ルール・施設一覧・マップ・バフ表 |
| 戦力分析 | `/[locale]/members/goldvein-prep/analysis` | 英雄/連盟の戦力データ・フィルタ・ヒストグラム |
| 戦略シート | `/[locale]/members/goldvein-prep/sheet` | 勝利条件・施設初期値・勝利シナリオ |
| シミュレーター | `/[locale]/members/goldvein-prep/simulator` | 占領時間計算（砲塔・攻城強化装置・バフ対応） |
| 戦略の考え方 | `/[locale]/members/goldvein-prep/strategy` | シナリオ分析・攻撃/防衛の勝利戦略 |
| ToDo | `/[locale]/members/goldvein-prep/discussion` | 確認事項・議論事項 |

### ログインページ

- URL: `/[locale]/goldvein-prep/login`
- ドラクエ風デザイン（バナー画像・RPG風ウィンドウ）
- `/goldvein-prep/` へのアクセスは自動でログインページにリダイレクト

### 多言語対応（6言語）

全ページを以下の6言語で翻訳:
- 日本語（ja）、English（en）、한국어（ko）
- 繁體中文（zh-TW）、简体中文（zh-CN）、العربية（ar）

ヘッダーに言語切替ボタンを設置。
用語辞書を `docs/services/terminology.md` に整備。

### Google Sheets連携

- 戦略シート: 施設Pt・勝利条件をCSV取得してテーブル表示
- 施設初期値・勝利シナリオ: iframe埋め込み表示
- 戦力分析: 英雄総戦力・連盟戦力をCSV取得
- Excelダウンロードボタン付き
- 言語フォールバック: 該当言語 → 英語 → 日本語

### 戦力分析機能

- Camp / Server / Tag の複数選択フィルタ（階層連動）
- Power範囲指定
- Defense / Attack 別の件数・平均戦力表示
- 戦力分布ヒストグラム（Defense青 / Attack赤で層別表示）

### シミュレーター

- 占領時間の計算（3フェーズ: 0-60秒・60-120秒・120秒以降）
- 攻城強化装置の60秒後取り消しロジック対応
- 施設プリセット（小型拠点〜ゴールデンパレス）
- 陣営切替・バフ選択
- 進行度ごとの占領時間テーブル（秒 + x分x秒表示）

### ディレクトリ構成（複数イベント対応）

```
app/[locale]/(members)/members/
├── goldvein-prep/        ← 今回のイベント
│   ├── guide/
│   ├── rules/
│   ├── analysis/
│   ├── sheet/
│   ├── simulator/
│   ├── strategy/
│   └── discussion/
└── （次回イベント）/      ← 同じ構造で追加可能
```

## 環境変数（追加分）

```bash
PREP_SPREADSHEET_FILE_ID=    # スプレッドシートファイルID
PREP_SPREADSHEET_PUB_ID=     # スプレッドシート公開ID
PREP_GID_JA_FACILITY=        # JA: 施設のPt
PREP_GID_JA_CONDITION=       # JA: 勝利条件
PREP_GID_JA_SCENARIO=        # JA: 勝利シナリオ
PREP_GID_EN_FACILITY=        # EN: 施設のPt
PREP_GID_EN_CONDITION=       # EN: 勝利条件
PREP_GID_EN_SCENARIO=        # EN: 勝利シナリオ
PREP_GID_JA_INITIAL=         # 施設の初期値
PREP_GID_HERO_POWER=         # 英雄総戦力
PREP_GID_ALLIANCE_POWER=     # 連盟戦力
```
