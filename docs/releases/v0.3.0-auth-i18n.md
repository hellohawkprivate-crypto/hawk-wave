## 概要

3層のアクセス制御と6言語対応を実装した。

---

## 実施内容

### 認証（3層アクセス制御）

| ページ | URL | 認証方式 |
|---|---|---|
| 公開 | `/[locale]` | なし |
| メンバー | `/members/strategy` | パスワード + Cookie |
| スタッフ | `/staff/analytics` | Discord OAuth |

**実装ファイル**
- `src/middleware.ts` — ルート保護（認証 + ロケール処理）
- `src/lib/auth.ts` — NextAuth設定（Guild+Role OR 許可リスト）
- `src/lib/sheets.ts` — Google SheetsからID許可リストを取得
- `src/app/(public)/login/page.tsx` — パスワードログインフォーム
- `src/app/api/auth/login/route.ts` — パスワード検証API
- `src/app/api/auth/[...nextauth]/route.ts` — NextAuthハンドラー
- `src/app/(members)/members/strategy/page.tsx` — メンバーページ
- `src/app/(staff)/staff/analytics/page.tsx` — スタッフページ

### スタッフID許可リスト（Google Sheets管理）

- Google Sheetsで `discord_id / server / alliance / name / memo` 列で管理
- シート更新のみでコード変更・再デプロイ不要
- 60秒キャッシュで毎回取得を避ける
- Guild + Role チェックとの併用（どちらかを満たせばOK）

### 多言語対応（next-intl v4）

| ロケール | 言語 | URL |
|---|---|---|
| `ja` | 日本語（デフォルト） | `/ja` |
| `en` | English | `/en` |
| `ko` | 한국어 | `/ko` |
| `zh-TW` | 繁體中文 | `/zh-TW` |
| `zh-CN` | 简体中文 | `/zh-CN` |
| `ar` | العربية（RTL対応） | `/ar` |
