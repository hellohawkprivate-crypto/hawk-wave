# public/

このフォルダには静的アセットを配置します。

- `favicon.ico` — サイトアイコン
- `images/` — 画像ファイル（マップ、施設画像等）
- `og-image.png` — OGP用画像（SNSシェア時のサムネイル）

## 画像ファイルの命名規則

```
facilities/
  small-base.png      # 小型拠点
  large-city.png      # 大都市
  medium-base.png     # 中型拠点
  fortress.png        # 軍事要塞
  golden-palace.png   # ゴールデンパレス

map/
  kinmyaku-map.png    # フィールドマップ全体
```
